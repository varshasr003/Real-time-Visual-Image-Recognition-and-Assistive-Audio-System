# Visual Object Detection System

## Description
This project is designed to detect objects in real-time and extract text using OCR. The system also converts the output into audio for assistive purposes.

## Technologies Used
- YOLO (Object Detection)
- OCR (Text Recognition)
- Python
- OpenCV

## Features
- Real-time object detection
- Text extraction from images
- Audio output generation
